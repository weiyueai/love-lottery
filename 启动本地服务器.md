# 🚀 启动本地服务器

## ❗ 重要说明

**不能直接双击打开 `index.html` 或 `login.html`！**

因为使用了ES6模块（import/export），必须通过HTTP服务器访问。

---

## 🎯 快速启动（推荐方法）

### 方法一：使用Python（最简单）

#### 如果有Python 3：
```bash
# 1. 打开命令提示符（cmd）
# 2. 进入项目文件夹
cd "D:\Users\yuew3\Desktop\礼物盒子"

# 3. 启动服务器
python -m http.server 8000
```

#### 如果有Python 2：
```bash
python -m SimpleHTTPServer 8000
