🚀 GitHub Pages 快速部署指南（5分钟完成）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📝 步骤1：创建GitHub账号（如果还没有）
   访问：https://github.com
   点击 Sign up 注册

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📦 步骤2：创建新仓库
   1. 登录后，点击右上角 + 号
   2. 选择 "New repository"
   3. 填写：
      Repository name: love-lottery
      选择: Public ✅
   4. 点击 "Create repository"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📤 步骤3：上传文件
   1. 点击 "uploading an existing file"
   2. 拖拽以下文件到上传区域：
      
      必须上传的文件（10个）：
      ✅ index.html
      ✅ history.html
      ✅ style.css
      ✅ script.js
      ✅ 描绘画面 (1).png
      ✅ 描绘画面 (2).png
      ✅ 描绘画面 (3).png
      ✅ 描绘画面 (5).png
      ✅ 补给站.jpg
      ✅ Hv1p6wAtu6hIOGXBjtzbo.png
   
   3. 提交信息填写：恋爱抽奖系统
   4. 点击 "Commit changes"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚙️ 步骤4：启用GitHub Pages
   1. 在仓库页面，点击 "Settings"（设置）
   2. 左侧菜单点击 "Pages"
   3. Source 选择：Deploy from a branch
   4. Branch 选择：main + / (root)
   5. 点击 "Save"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⏰ 步骤5：等待部署（1-3分钟）
   - 页面会刷新显示部署状态
   - 看到绿色提示后，复制网址

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🌐 你的网站地址：
   https://你的用户名.github.io/love-lottery/

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 常见问题：

Q: 网站打不开？
A: 等待3-5分钟，清除浏览器缓存（Ctrl+F5）

Q: 图片不显示？
A: 确保所有图片都已上传，文件名没有错误

Q: 如何更新内容？
A: 在GitHub仓库页面，点击文件→编辑→保存
